Before starting this workshop, you should prepare the required cloud resources, local development tools, and project source code. The following prerequisites are recommended.

### 1. Cloud Account & AWS Access

- **AWS Account:** An active AWS account with sufficient permissions to create and manage AWS resources.
- **Core AWS Services:** Access to the AWS services used throughout the project, including AWS Lambda, Amazon API Gateway, Amazon DynamoDB, Amazon Cognito, Amazon S3, Amazon CloudFront, and AWS WAF.

### 2. Local Development Environment

- **Node.js & npm:** Install the latest **LTS** version of Node.js to ensure full compatibility with the AWS Lambda runtime environment.
- **Git & GitHub:** Install **Git** for source code management and use a **GitHub** account to host and collaborate on the project repository.
- **AWS CLI (Command Line Interface):** Install the AWS CLI and run the `aws configure` command to connect your local machine to your AWS account using your Access Key credentials.
- **Integrated Development Environment (IDE):** Install **Visual Studio Code (VS Code)** as the primary code editor for developing and managing the project.
