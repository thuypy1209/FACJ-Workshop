Để thực hiện thành công workshop này, bạn cần chuẩn bị sẵn sàng các tài nguyên đám mây, công cụ phát triển dưới máy cục bộ (Local) và mã nguồn của dự án. Dưới đây là các điểm chính cần chuẩn bị

### 1 Tài khoản đám mây & Quyền truy cập (AWS Cloud Setup)
-Tài khoản AWS: Một tài khoản AWS đang hoạt động
-các công cụ chính dùng để xây lên dự án 

### 2. Môi trường phát triển cục bộ
-Node.js & NPM: Cài đặt Node.js phiên bản LTS  nhằm đảm bảo tính tương thích tuyệt đối với môi trường Runtime của AWS Lambda Backend.
- Git & GitHub: * Cài đặt Git dưới máy để quản lý mã nguồn.
-AWS CLI (Command Line Interface): Cài đặt công cụ AWS CLI dưới máy và chạy lệnh aws configure để liên kết máy tính với tài khoản AWS thông qua Access Key đã tạo.
-Trình soạn thảo mã nguồn (IDE): Visual Studio Code (VS Code)
