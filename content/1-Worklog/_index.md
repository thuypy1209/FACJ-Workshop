**On this page**, overview of worklog throughout the internship. The internship lasted **12 weeks**, from 05/05/2026 to 25/07/2026, during which I researched AWS Serverless services and used them to design, build, and deploy an **Online Tour Booking System**, covering the full architecture from Edge Security & Delivery (Route 53, WAF, CloudFront, S3 Frontend) to the Serverless backend (API Gateway, Cognito, Lambda, S3 Storage, DynamoDB) and monitoring (CloudWatch).

Below is a weekly breakdown of the worklog:

**Week 1:** [Getting acquainted with AWS Serverless and surveying the tour booking system's requirements](1.1-week1/)

**Week 2:** [System Architecture Design (Route 53, WAF, CloudFront, API Gateway, Cognito, Lambda, S3, DynamoDB, CloudWatch)](1.2-week2/)

**Week 3:** [NoSQL Database Design with Amazon DynamoDB](1.3-week3/)

**Week 4:** [User Authentication with Amazon Cognito (User Pool & Identity Pool)](1.4-week4/)

**Week 5:** [Serverless Compute with AWS Lambda (CRUD Logic & JWT Verification)](1.5-week5/)

**Week 6:** [API Gateway, WAF Protection & Cognito Authorizer](1.6-week6/)

**Week 7:** [Frontend Project Initialization with ReactJS/Vite](1.7-week7/)

**Week 8:** [Frontend Authentication Feature (Login/Register with JWT)](1.8-week8/)

**Week 9:** [Tour Listing & Booking Request Feature](1.9-week9/)

**Week 10:** [UI/UX Polishing, Responsive Design & Frontend Deployment](1.10-week10/)

**Week 11:** [End-to-End System Testing & Performance Optimization](1.11-week11/)

**Week 12:** [Final Internship Report & Technical Documentation](1.12-week12/)
